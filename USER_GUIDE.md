# 用户操作手册草稿

本文档面向日常使用者，说明第一版 MVP 的主要操作流程。

## 1. 基础资料录入顺序

建议按以下顺序录入：

1. 客户管理
2. 客户项目
3. 合同管理
4. 员工管理
5. 账户管理
6. 系统设置中的收支分类
7. 财务管理中的收支流水

先录基础资料，再录流水，后续项目统计和报表才能正确关联。

## 2. 客户管理

进入“客户管理”页面，可以新增、编辑、删除客户。

客户删除为软删除，不会从数据库中物理移除。

## 3. 客户项目

进入“客户项目”页面，可以维护客户对应的项目。

项目必须选择所属客户。

项目列表会显示：

1. 合同金额
2. 已收款
3. 已支出
4. 应收款
5. 当前毛利
6. 预计毛利
7. 收款状态

点击“统计”可以查看项目详情，包括合同列表和相关流水。

## 4. 合同管理

进入“合同管理”页面，可以录入客户项目合同。

合同金额在页面输入为元，系统内部保存为整数分。

合同必须关联客户和项目，且项目必须属于所选客户。

点击合同列表中的“附件”可以维护合同附件。

当前支持：

1. 上传 JPG、PNG、WEBP 图片。
2. 上传 PDF 文件。
3. 对图片和 PDF 附件手动上移、下移排序。
4. 将当前合同下的图片按排序生成一个 PDF 文件。
5. 在系统内预览图片和 PDF。
6. 使用系统默认工具打开已上传或已生成的 PDF。
7. 重命名附件显示名称。
8. 查看附件大小、创建时间和本地文件状态。

附件删除为软删除，文件会保留在本地数据目录中，列表中不再显示。

如果附件对应的本地文件被手动删除，系统会显示“本地文件丢失”，并禁用预览和打开操作。

## 5. 财务管理

进入“财务管理”页面，可以录入收入和支出。

流水需要选择：

1. 发生日期
2. 收支分类
3. 方向
4. 金额
5. 账户
6. 资金性质

可选关联：

1. 客户
2. 项目
3. 员工

选择收支分类后，系统会自动带出方向、资金性质、是否影响应收、是否影响项目利润。

## 6. 作废流水

财务流水不得物理删除。

如果一笔流水录错，优先使用“作废”，并填写作废原因。

作废流水不会参与：

1. 账户余额
2. 项目统计
3. 报表统计

## 7. 报表对账

进入“报表对账”页面，可以查看：

1. 月度收支表
2. 项目利润表
3. 客户应收表
4. 账户余额表

报表金额显示为元。

## 8. 首页仪表盘

首页展示当前经营概览：

1. 本月收入
2. 本月支出
3. 本月收支差额
4. 账户总余额
5. 项目应收合计
6. 项目预计毛利合计

## 9. 数据备份

进入“系统设置”页面，点击“备份数据库”。

系统会生成 SQLite 备份文件，并提示备份路径。

建议在大量录账后、月底对账后、导出报表前手动备份。

## 10. Excel 导出

进入“系统设置”页面，点击“导出 Excel”。

导出内容包括：

1. 客户
2. 项目
3. 合同
4. 财务流水
5. 月度收支表
6. 项目利润表
7. 客户应收表
8. 账户余额表

Excel 中金额字段以元展示。

## 11. 暂不支持的功能

当前版本暂不支持：

1. 自动恢复数据库
2. 附件拖拽排序
3. 多账套
4. 权限系统
5. 云同步
6. 自动更新
7. 正式签名安装包
